estas son las herramientas de desarrollo de software


Visual Studio proporciona a los desarrolladores un entorno de desarrollo enriquecido para desarrollar código de alta calidad de forma eficaz y colaborativa. Python es un lenguaje de programación ampliamente utilizado en las aplicaciones web, el desarrollo de software, la ciencia de datos y el machine learning (ML). JavaScript es un lenguaje de programación que los desarrolladores utilizan para hacer páginas web interactivas. Java es un lenguaje de programación multiplataforma orientado a objetos que se ejecuta en miles de millones de dispositivos de todo el mundo.
Visual studio python javascript java
El HTML5 es un estándar que sirve para definir la estructura, el diseño y el contenido de una página web. GitHub es una plataforma basada en la nube donde puedes almacenar, compartir y trabajar junto con otros usuarios para escribir código. El CSS podría definirse como un tipo de lenguaje que permite definir y crear la presentación visual de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML. C++ es un lenguaje de programación orientado a objetos muy potente que evolucionó de la extensión de lenguaje informático.
